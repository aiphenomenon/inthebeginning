## Architectural Decisions

### 1. Tempo Range: 1.2x–1.8x (narrower, lower than v9)
- v9 had 1.1x-2.1x with density-aware capping
- v10 locks it to 1.2x-1.8x flat — no density-dependent capping
- This gives a consistently moderate pace that avoids both sluggish and frantic extremes

### 2. Instrument Diversity — Fix the "Organ-y" Problem
- Root cause: v7/v8/v9 all use `_choose_gm_instruments` which picks from family pools
  BUT the *base note synthesis* (`synthesize_colored_note`) uses the `InstrumentFactory`
  which generates ~537 synthetic instruments at init. These are additive synth patches
  that all sound somewhat organ-like because they share similar harmonic structures
- Fix for v10:
  - Use GM program number to *influence the timbre* of synthesized notes
  - Map GM families to distinct synthesis parameters (attack, decay, harmonic ratios)
  - GM piano (0-7): short attack, fast decay, bell-like harmonics
  - GM strings (40-51): slow attack, sustained, odd harmonics
  - GM brass (56-63): medium attack, bright, strong 2nd/3rd harmonic
  - GM woodwinds (64-79): soft attack, pure tone, hollow
  - GM synth leads (80-87): instant attack, sawtooth-like
  - GM synth pads (88-95): very slow attack, deep sustain
  - GM world (104-111): varied, often plucked (fast attack, moderate decay)
  - GM mallets (9-15): percussive, very fast attack+decay
  - This breaks the organ-y monotone by giving each GM program a distinct envelope/color

### 3. Orchestral Sections — Simultaneous Majority
- v8 already has 75% simultaneous / 25% solo
- v10 raises to 85% simultaneous / 15% solo
- Add 4-6 voices (up from 3-5) for richer orchestral texture
- Ensure at least 3 different family groups per segment for true "sections" feel
- Register spread: bass (-24, -12), mid (0), treble (+12, +24) for fuller separation

### 4. Mood Duration — Multiples of 42s (Same as v9)
- Keep the same segment planning as v8/v9
- Duration multiples: [42, 84, 84, 126, 126, 168, 210]

### 5. TTS Announcements
- Keep existing TTS injection mechanism (once per 10-min window)
- Ensure speech includes source code phrases AND "in the beginning radio"
- Already implemented in `_inject_tts_at` — reuse as-is

### 6. Improved Fades Between Moods
- Increase MORPH_DURATION from 6.0s to 8.0s (33% longer crossfade)
- Use double-cosine morph curve for smoother perceptual transition
- Add a subtle reverb tail during morph to fill the gap

### 7. Two MP3s
- `cosmic_radio_v10.mp3` with seed=42
- `cosmic_radio_v10_random.mp3` with seed=random.randint(1, 999999)
- Duration: 1800s (30 minutes) each
